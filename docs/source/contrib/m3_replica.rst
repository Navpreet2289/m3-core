m3_replica - репликация данных
==============================

Приложение ``m3.contrib.m3_replica`` содержит функционал, позволяющий разрабатывать
подсистемы импорта/эспорта данных в прикладных проектах.

Любая операция по импорту/экспорту данных рассматривается как процесс передачи
серии порций данных от источника данных (**DataSource**) приемнику данных 
(**DataTarget**). Процесс передачи данных управляется объектом класса 
**DataExchange**.
