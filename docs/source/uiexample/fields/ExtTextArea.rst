ExtTextArea
------------

    .. autoclass:: m3.ui.ext.fields.simple.ExtTextArea

    .. image:: /images/ui-example/textarea.png

Является потомком ``BaseExtField``. Обертка `Ext.form.TextArea <http://docs.sencha.com/ext-js/3-4/#!/api/Ext.form.TextArea>`_
